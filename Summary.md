# CS50
## Definitions 
Unary - base 1 ie. repeated use of one symbol

Dec - ten digits ie. 0 to 9

Binary - base 2 ie. use of two symbols and accounts for position

Bit - binary digit ie. 0 or 1

Byte - 8 bits


## Binary System
<mark>Maximum number you can count up to with n transmittors = 2<sup>n</sup> -1\
eg. With 3 transmitters, count up to 7 hence 8 possibilities</mark> 

Decipher a binary number by multiplying each digit(0 or 1) with its corresponding column(powers of 2)\
eg. 1001(binary) = 9(decimal) = 1x2<sup>3</sup> + 0x2<sup>2</sup> + 0x2<sup>1</sup> + 1x2<sup>0</sup>
